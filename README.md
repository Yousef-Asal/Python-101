# Python-101

This repo is the application for the course **Python for pragramming and Computer Vision** for the masters program **Mobile Robotics** in **Bonn University**.
The structure of this repo is fairly simple. you will find:

- Main Resource: [Introduction to Programming with python (book)](<Introduction to Programming with Python.pdf>)
- Practice: [Book Practicals](Practicals.ipynb)
- Scattered files related to the practicals (Ch9)
- Image folders that contain images for ch12 practicals and output images.
- Assignment folders, with each one have a notebook containing the questions and the solution for each question in its respective file.
